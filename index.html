Malocher Framework

CQRS + Service Bus library for PHP projects

About Malocher

The Malocher Framework implements the service bus architecture and Command Query Responsibility Segregation (CQRS) principles for php applications. CQRS plays well with Domain Driven Design (DDD) or traditional MVC patterns.

2013/12/10 - First stable release

The first stable release of the Malocher CQRS and Enterpise Service Bus library is available. You can make use of features like activating multiple buses in your system and subscribing to meta messages on the system bus to monitor your entire CQRS message flow.
